# **Ubuntu Shortcut cheat Sheet**

```
1.Windows + c   # Chrome open.
2.Windows + T   # Terminal open.
3.Windows + E   # Text editor.
4.Alt + Tab     # Switch Application.
5.ctrl + Q      # clean ubuntu.
6.Windows +     # Minimize Application.
7.Windows+(pgup/pgdn) # Application zoom in out. 
```

# **Chrome Shortcut cheat Sheet**

```
1. ctrl + T         # New Tab open.
2. ctrl + W         # Closed Tab.
3. Windows + H      # Minimize. 
4. ctrl + Tab       # Change Tab.
5. ctrl + (pgup/pgdn) # Change Tab.
```



# **CMD kyebord shortcut**

```
1. Touch filename.txt   # File create.
2. mkdir Folder name    # Folder create.
3. cd                   # change directory. 
4. ls                   # Show all the list and Folder.
5. clear                # clear all cmd
6. ctrl + Q             # quit 
```


# **VS-Code kyebord shortcut**

```
1. code .       # open VS- code.
2. ctrl + j     # Terminal (open / hidden).
3. ctrl+shift+E  # activity bar.
4. ctrl + B     # Activity bar hidden.
5. ctrl + (pgup/pgdn) # Change Tab
6. B pas        # new cell.
7. D pas        # cell delete.
8. ctrl + z     # undo. 
9. shift+Enter  # Run Program.

```
