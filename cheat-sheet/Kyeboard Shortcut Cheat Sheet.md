# **Keyboard Shortcut**

```
Here is list of common applications:
1. Ubuntu
2. Chrome Browser
3. VS-Code
4. CMD|Terminal
5. Git & GitHub
```


## **Ubuntu**
```
1. Windows + C          # Chrome Open
2. Windows + T          # Terminal Open
3. Windows + E          # Text Editor
4. Alt + Tab            # Switch Application
5. Ctrl + Q             # Close Application (Except CMD|Terminal)
6. Windows + H          # Minimize Application.
7.Windows + Page Up     # Application Zoom In out
8. Windows + Page Down  # Application Zoom Out
```

## **CMD|Terminal**
```
1. touch filename.txt   # File create
2. mkdir folder_name    # Folder create
3. cd path/             # change directory 
4. ls                   # Show all file and folder as list
5. clear                # Clear all cmd
6. exit                 # Close terminal 
```

## **Chrome Browser**
```
1. Ctrl + T                     # New Tab Open
2. ctrl + W                     # Closed Tab
3. Windows + H                  # Minimize 
5. Ctrl + Page Up|Page Down     # Change Tab
```


## **VS-Code**
```
1. code .                       # Open VS- code.
2. Ctrl + j                     # Terminal open|hide
3. Ctrl + Shift+E               # Activity bar file access
4. Ctrl + B                     # Activity bar open|hide
5. Ctrl + Page Up|Page Down     # Change work space tab
6. Shift+Enter                  # Run Python on Jupyter notebook
7. B                            # New cell on Jupyter notebook
8. D                            # Delete cell on Jupyter notebook
9. Ctrl + Z                     # Undo 
```


## **Git & GitHub**

```
list of steps to follow GitHub (The-A-Team) repository:
1. Do your job (New file / edit file )
2. Open cmd on root.
3. Update repository
        $ git pull
3. Copy work File or edit the work file
4. Write commit message on commit_tracing.text
5. Upload on github with this commands 
    $ git add .
    $ git commit -m "message"
    $ git push
6. Check your work on GitHub.
7. Message new commit have done on Whatsapp.
```
