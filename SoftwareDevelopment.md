What is Software Development?
Software Development is the process of taking set of requirements from user, analyze them, design solution and implements the solution on computer.

The challenge of software development is every requirements is new and you probably not work before. So, the core skills and experience (mental model) is key here.


How do you train your brain for Software Development?
- Doing hands on live projects
- Understanding lots of client requirements on Upwork feed
- Reading good design materials
- Preview lots of code
- Consuming programming YouTube content
- Reading programming and business books


What are steps to follow building a Software Project?
1. Client communication & requirements engendering
2. SRS Document
3. Design Document
4. Coding
5. Testing
6. Deployment
7. Maintenance

Today modern software project follow Agail away (very good process). Here is the philosophy about Agail:
1. Motivated individuals are important rather then process and tools
2. Working software over comprehensive documentations
3. Customer collaborations over contract paper
4. Responding change over following a plan
5. Highest priority is to satisfy customer through early and continuous delivery.


I slightly modified it based on my freelance projects. Here is steps to follow building a freelance software project:
1. Client message requirements (through Fiverr, LinkedIn or Upwork)
2. Client communications and asking important questions
3. BrainStorm client messages and building a format SRS document
4. Building minimal Design document
5. Hard code, cover as much as default features and send to client
6. Getting feedback and mature the project over time
7. Asking client pay me every new feature
